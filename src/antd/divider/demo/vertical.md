## zh-CN

使用 `type="vertical"` 设置为行内的垂直分割线。

## en-US

Use `type="vertical"` make it vertical.

```jsx
import { Divider } from 'antd';

ReactDOM.render(
  <>
    Text
    <Divider type="vertical" />
    <a href="#">Link</a>
    <Divider type="vertical" />
    <a href="#">Link</a>
  </>,
  mountNode,
);
```
